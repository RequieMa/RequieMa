系统并不等同于部分之和 -- 信号与噪声交叠传播，有序与混沌共存，整体总是令人始料未及。

控制论 (Cybernetics) 曾指向一门足以应对日益增加的复杂性的新科学。但那个愿景被搁置了太久。我在这里呼唤那些同样关注它的人们。
